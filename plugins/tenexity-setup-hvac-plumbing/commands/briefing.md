# /briefing — On-Demand Morning Briefing

Run the daily-briefing skill immediately, regardless of whether the scheduled task has run today.

Read `${CLAUDE_PLUGIN_ROOT}/skills/daily-briefing/SKILL.md` and follow the "Morning briefing" section. Use the same format and data sources as the scheduled version — including the distributor-specific AR watch and vendor watch sections.

This command exists because users sometimes want their briefing at a different time, or want a refresh after making changes. Treat it identically to the scheduled morning briefing.
